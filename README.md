# TalesOfCyber

## Basic Commands

Start in Dev Mode

```bash
npm start
```

Start in Dev Mode and make accessable in local Network

```bash
ng serve --host 0.0.0.0
```

Check Packages for vulnerabilities

```bash
npm audit
```
